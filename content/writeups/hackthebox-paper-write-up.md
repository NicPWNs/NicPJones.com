<p>This is my write-up for the Paper machine on HackTheBox that just retired! Here I detail the penetration testing steps taken to scan, exploit, and privilege escalate on this target machine. This box is special because it makes use of an exploit I’ve developed in my <a href="https://unicord.dev/">UNICORD</a> project. This machine is categorized as <em>easy</em> difficulty and was retired on June 18th, 2022.</p><h2>Paper Summary</h2><figure><img alt="" src="/writeups/hackthebox-paper-write-up/img-1.png"  decoding="async" width="300" height="300" loading="eager" fetchpriority="high" /></figure><h2>Target Information</h2><p><a href="https://app.hackthebox.com/machines/paper"><strong>Machine Page</strong></a><br><strong>IP Address:</strong> 10.10.11.143<br><strong>Hostname:</strong> paper.htb</p><h2>Synopsis</h2><p>A hidden hostname must be discovered to access a WordPress site running on the target. The WordPress core version is vulnerable to disclose drafts from editors that disclose another hidden hostname and website. The next website contains an open chat application with a bot that can be used to disclose files, including a password. This password is reused to allow for SSH access. The system is vulnerable to a policy kit CVE that allows for the creation of a new sudo user for full root access.</p><h2>Scanning</h2><h2>Nmap</h2><p>Nmap scans show SSH open on port 22/tcp, HTTP open on port 80/tcp, and HTTPS open on port 443/tcp.</p><pre># nmap -sV -sC -p- 10.10.11.143<br>Starting Nmap 7.92 ( https://nmap.org ) at 2022-06-19 22:47 GMT<br>Nmap scan report for 10.10.11.143<br>Host is up (0.061s latency).<br>Not shown: 65532 closed tcp ports (reset)<br>PORT    STATE SERVICE  VERSION<br>22/tcp  open  ssh      OpenSSH 8.0 (protocol 2.0)<br>| ssh-hostkey: <br>|   2048 10:05:ea:50:56:a6:00:cb:1c:9c:93:df:5f:83:e0:64 (RSA)<br>|   256 58:8c:82:1c:c6:63:2a:83:87:5c:2f:2b:4f:4d:c3:79 (ECDSA)<br>|_  256 31:78:af:d1:3b:c4:2e:9d:60:4e:eb:5d:03:ec:a0:22 (ED25519)<br>80/tcp  open  http     Apache httpd 2.4.37 ((centos) OpenSSL/1.1.1k mod_fcgid/2.3.9)<br>|_http-generator: HTML Tidy for HTML5 for Linux version 5.7.28<br>| http-methods: <br>|_  Potentially risky methods: TRACE<br>|_http-title: HTTP Server Test Page powered by CentOS<br>|_http-server-header: Apache/2.4.37 (centos) OpenSSL/1.1.1k mod_fcgid/2.3.9<br>443/tcp open  ssl/http Apache httpd 2.4.37 ((centos) OpenSSL/1.1.1k mod_fcgid/2.3.9)<br>|_http-title: HTTP Server Test Page powered by CentOS<br>| http-methods: <br>|_  Potentially risky methods: TRACE<br>|_http-generator: HTML Tidy for HTML5 for Linux version 5.7.28<br>| tls-alpn: <br>|_  http/1.1<br>| ssl-cert: Subject: commonName=localhost.localdomain/organizationName=Unspecified/countryName=US<br>| Subject Alternative Name: DNS:localhost.localdomain<br>| Not valid before: 2021-07-03T08:52:34<br>|_Not valid after:  2022-07-08T10:32:34<br>|_ssl-date: TLS randomness does not represent time<br>|_http-server-header: Apache/2.4.37 (centos) OpenSSL/1.1.1k mod_fcgid/2.3.9</pre><pre>Service detection performed.
